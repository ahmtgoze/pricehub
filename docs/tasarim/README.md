## Özet

- **Referans:** Apple.com estetiği — minimal, sakin, monokrom
- **Renkler:** Beyaz / açık gri arka plan, siyah aktif elementler, kırmızı/yeşil sadece anlam taşıdığında
- **Font:** System font stack (SF Pro / Inter), SF Mono sayılar için
- **Kart:** `bg-white rounded-2xl border border-gray-100 shadow-sm`
- **Buton:** `bg-gray-900 hover:bg-gray-800` (birincil), ghost (ikincil)
